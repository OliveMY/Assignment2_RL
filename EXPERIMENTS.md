### PPO Diagnostics Glossary
| Metric | What It Measures |
|--------|-----------------|
| Clip Fraction | Fraction of policy updates that hit the clipping bound. High = policy trying to change faster than clip_range allows. |
| Approx KL | KL divergence between old and new policy. High = large policy updates per step. |
| Entropy Loss | Negative entropy of the policy. Decreasing entropy = agent becoming more deterministic/confident. |
| Explained Variance | How well the value function predicts returns. 1.0 = perfect, <0 = worse than mean prediction. |
| Policy Gradient Loss | The PPO surrogate objective. More negative = larger policy improvement. |
| Value Loss | MSE of value function predictions. Higher during active learning, lower when stable. |
